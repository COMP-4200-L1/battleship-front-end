SOUND FILES

Place the following sound files in this directory:

1. explosion.mp3 - Sound effect for when a ship is hit
2. splash.mp3 - Sound effect for when a shot misses
3. ship_place.mp3 - Sound effect for when a ship is placed
4. ship_rotate.mp3 - Sound effect for when a ship is rotated
